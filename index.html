What’s new in v2.0

Released June 2026 · Release notes on GitHub

  • Proxyman-style sidebar — Domains, Devices, and Pinned sections with per-domain API endpoints
  • Browse while capturing — Chrome, Safari, and other apps keep working; HTTPS tunnels correctly
  • macOS Wi‑Fi proxy — auto-enable on Start and restore on Stop / Quit
  • Selective SSL decrypt — pages load by default; decrypt only the hosts you choose
  • Quit confirmation — stops capture and restores system proxy before the app closes
  • User-pinned domains — your pins persist until you remove them
  • In-memory capture — session data clears when you quit (nothing stored on disk)

Get started in minutes — for free

Download the DMG, drag the app to Applications, and start capturing traffic. No account, no license key, no credit card.

1

Download

Grab the latest ShubhranshProxy v2.0 DMG for macOS.

2

Install

Open the DMG and drag the app into Applications.

3

Open ShubhranshProxy and click Start.

4
